# Contributing Guidelines

Thank you for your interest in contributing to this project   
This is an open-source project, but it follows a **clear ownership and governance model**.

---

##  Project Ownership

- The repository owner is **@praakhartripathi**
- Only the owner has the authority to:
  - Approve pull requests
  - Merge code into the `main` branch
  - Decide which features are included in the product

---

##  How You Can Contribute

You are welcome to:
- Report bugs using GitHub Issues
- Suggest new features or improvements
- Fork the repository
- Create feature branches
- Submit pull requests (PRs)

---
